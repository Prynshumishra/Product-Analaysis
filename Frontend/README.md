# Frontend - Interactive Product Analytics Dashboard

This folder contains the React + Vite frontend for the dashboard.

For full architecture, backend setup, API docs, seeding, deployment, and scaling strategy, see the repository root README at:

- ../README.md

## Local Commands

1. Install dependencies:
	npm install
2. Configure environment:
	copy .env.example .env
3. Run development server:
	npm run dev
4. Build for production:
	npm run build
